AI Home Project

Developer: Mahendra Mali, Gujarat

Overview

The AI Home Project is a smart home automation system designed to control 50 relays using an Arduino Mega and an ESP-01 module. The system utilizes JSON data transmitted via serial communication to manage various appliances connected to the relays.

Components

Arduino Mega, ESP-01 Module, 50 Relays, EEPROM

Connections

Relays: Connected to Arduino pins 22 through 71. ESP-01 Connections: RX -> Arduino Mega TX1 (pin 18), TX -> Arduino Mega RX1 (pin 19)

Key Points

  • Relay Control: The system controls 50 relays based on data received from the ESP-01.
  • EEPROM Usage: The state of each relay is stored in EEPROM, allowing the system to remember the last state of each relay even after a power cycle.
  • Active LOW Relays: The relays are assumed to be active LOW, meaning they turn on when the pin is set to LOW.
  • Serial Communication: The ESP-01 module communicates with the Arduino Mega via the Serial1 interface, sending JSON formatted data to control the relays.
